CSS STYLING DEMONSTRATION

CSS(Cascading Style Sheets) is a language in web development to style and position elements of html files. There are many ways to include CSS code in a website.

  1. Inline CSS : This is a very straightforward way of injecting CSS code in an element of html file. CSS code in written as style attribute of the element. This type of CSS code has the highest precedence as it is injected directly into the html element.

  2. Internal CSS : This can be used when a single HTML document must be styled uniquely. The CSS rule set should be within the HTML file in the head section i.e. the CSS is embedded within the HTML file.

  3. External CSS : External CSS contains separate CSS file which contains only style property with the help of tag attributes (For example class, id, heading, â€¦ etc.). CSS property written in a separate file with .css extension and should be linked to the HTML document using link tag. This means that for each element, style can be set only once and that will be applied across web pages.
